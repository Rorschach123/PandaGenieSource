模块概述 Module overview

密码生成器（password_gen） 使用 java.security.SecureRandom 生成加密安全的随机密码与助记短语（英文词表）， 并对任意字符串进行启发式 password strength 分析（评分 0–100、rating、熵估计、checks、recommendations）。 展示文案随系统默认语言在中文/英文间切换（isZh()）。

Capabilities： clipboard（提示用户复制，逻辑不直接写剪贴板）。

架构 Architecture

  • 字符池：UPPER、LOWER、DIGITS、SYMBOLS。
  • generatePassword：每种勾选类型先 强制保证至少一个字符，再用合并池填满剩余位，最后 shuffleChars（Fisher–Yates）打乱。
  • 若四类全为 false，则回退到至少使用小写池，避免空池。
  • length 默认 16，clamp 到 4–256；generateMultiple 的 count 默认 5，范围 1–50。
  • generatePassphrase：wordCount 默认 4，范围 2–16；从 WORDS 数组随机选词，优先去重，不足则允许重复补齐。

强度分析 analyzeStrength

输出字段包括：score、rating（weak/fair/good/strong/excellent）、ratingLabel、entropyBits、 checks（length, mixedCase, numbers, symbols）、recommendations、hasLower/hasUpper/hasDigit/hasSymbol、length。

评分与熵

长度阶梯加分、字符类型加分、variety（类型数）额外加分；estimateEntropyBits 用池大小与长度近似 length * log2(pool)。

isSequentialOrRepeated：三连相同字符或三位连续递增/递减数字会 -15 分并推送建议。
