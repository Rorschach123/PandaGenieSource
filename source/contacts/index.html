联系人管理 / Contacts Manager

搜索、查看、导出联系人，查找重复联系人

功能接口

  • searchContacts — 按姓名或电话号码搜索联系人，返回 id、姓名、主号码；query 为关键词，limit 默认 20
  • getContactDetail — 按联系人 ID 返回完整信息：所有电话、邮箱、地址、公司职务等
  • listContacts — 分页列出联系人：id、姓名、主号码；limit 默认 50，offset 默认 0
  • getContactCount — 返回设备上联系人总数
  • exportContacts — 将全部联系人导出为 vCard 3.0（VCF）文件；outputPath 默认 /sdcard/PandaGenie/output/contacts_export.vcf
  • findDuplicates — 查找姓名相同或电话号码相同（规范化后）的重复联系人分组

使用方式

在对话中用自然语言描述你的需求，AI 会自动调用本模块的功能。
