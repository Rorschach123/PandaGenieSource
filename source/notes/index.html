笔记助手 / Notes

创建、编辑、删除、搜索和导出笔记

功能接口

  • createNote — 创建笔记，生成唯一 ID，以 JSON 保存至 /sdcard/PandaGenie/data/notes/{id}.json
  • listNotes — 列出所有笔记摘要（id、标题、创建/更新时间、正文预览）；可选关键词过滤标题与正文
  • getNote — 按笔记 ID 读取完整 JSON 内容
  • updateNote — 按 ID 更新笔记标题或正文；传入的字段才会被更新
  • deleteNote — 按 ID 删除对应 JSON 文件
  • searchNotes — 在标题与正文中按关键词搜索，返回匹配的笔记摘要列表
  • exportNote — 将指定笔记导出为纯文本文件；path 可为 .txt 完整路径或目录（目录下生成 {id}.txt）

使用方式

在对话中用自然语言描述你的需求，AI 会自动调用本模块的功能。
