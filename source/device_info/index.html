设备信息 / Device Info

查看设备型号、CPU、内存、存储、屏幕等系统信息

功能接口

  • getDeviceInfo — 获取品牌、型号、制造商、Android 版本、SDK 级别、构建号与系统指纹等基础信息，用户说【设备信息】【手机型号】【系统版本】时使用
  • getCpuInfo — 读取 /proc/cpuinfo 并返回处理器数量与架构等 CPU 信息，用户说【CPU】【处理器】【几核】时使用
  • getMemoryInfo — 通过 ActivityManager 获取内存总量、可用、已用与占用比例，用户说【内存】【运存】【RAM】时使用
  • getStorageInfo — 使用 StatFs 与 Environment 获取内部存储与外部主存储的总空间、可用与已用，用户说【存储】【空间】【硬盘】时使用
  • getDisplayInfo — 通过 WindowManager 与 DisplayMetrics 获取屏幕宽高、密度、densityDpi 与刷新率，用户说【屏幕】【分辨率】【刷新率】时使用
  • getSystemSummary — 一次调用汇总设备、CPU、内存、存储与屏幕信息，用户说【设备概况】【整机信息】时使用

使用方式

在对话中用自然语言描述你的需求，AI 会自动调用本模块的功能。
