链接解析 / Link Parser

解析网页、提取资源与元数据，并检查链接可用性。

需网络权限

功能特性

🌐网页解析
🖼图片提取
📥下载检测
🔗链接分类
📄正文提取
📋元数据分析
🔍URL 可用性
🔒SSL 证书

功能接口

  • parseUrl — 综合解析 URL，获取标题、描述、图片、链接、下载等
  • getHeaders — 获取 HTTP 响应头
  • extractImages — 提取页面内全部图片
  • extractLinks — 提取并分类超链接
  • extractDownloads — 检测可下载文件
  • extractText — 提取纯文本正文
  • checkUrl — 检查可访问性、重定向、SSL 证书等
  • extractMeta — 提取 Open Graph、Twitter Card 等元数据

使用示例

  • 解析这个链接 https://example.com
  • Parse this URL https://example.com
  • 检查这个网址是否可访问
  • 提取 https://example.com 的 Open Graph 与标题
