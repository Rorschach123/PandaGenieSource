模块概览

对本地图片进行 信息查询（inJustDecodeBounds + EXIF）、缩放、质量压缩、格式转换、旋转、区域裁剪（BitmapRegionDecoder）。输出路径由调用方指定或使用默认在文件名插入后缀。

manifest：file_read、file_write；MANAGE_EXTERNAL_STORAGE。pluginClass：ai.rorsch.moduleplugins.image_tools.ImageToolsPlugin。

解码与内存策略

  • MAX_DECODE_SIDE = 4096：computeInSampleSize 使用 2 的幂次 inSampleSize，避免 OOM。
  • decodeBounds：仅边界与 MIME。
  • loadBitmapForProcessing：全图处理前子采样解码。
  • cropImage：BitmapRegionDecoder.decodeRegion + 区域级 inSampleSize。
  • WebP：API 30+ 使用 WEBP_LOSSY / WEBP_LOSSLESS 分支（webpCompressFormat）。
  • PNG：jpegQualityForFormat 固定质量 100（无损）。

类 ImageToolsPlugin（均为 static 辅助，除接口方法）
