图片工具 / Image Tools

图片信息查看、缩放、压缩、格式转换、旋转、裁剪

功能接口

  • getImageInfo — 获取图片信息：宽高、格式、文件大小、EXIF（方向、日期、GPS 等）
  • resizeImage — 将图片缩放到指定宽高并保存
  • compressImage — 压缩图片以减小体积；quality 对 JPEG/WEBP 有效
  • convertFormat — 在 PNG、JPG、WEBP 之间转换格式
  • rotateImage — 按角度旋转图片（90/180/270）
  • cropImage — 按矩形区域裁剪图片（坐标相对原图）

使用方式
