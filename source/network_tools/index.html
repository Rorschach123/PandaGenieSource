网络工具 / Network Tools

Ping测试、DNS查询、IP地址、网络连接状态

功能接口

  • ping — Ping 指定主机（系统 ping 命令，默认 4 次），返回标准输出与退出码
  • dnsLookup — DNS 解析域名，返回解析到的所有 IP 地址
  • getLocalIp — 枚举网卡，获取本机非回环的 IPv4/IPv6 地址列表
  • getPublicIp — 通过 HTTPS 请求 api.ipify.org 获取当前公网出口 IP（需网络）
  • checkConnectivity — 使用 ConnectivityManager 检查当前网络是否可用及是否已验证连通
  • getNetworkInfo — 当前网络类型（WiFi/蜂窝/无等）、WiFi SSID（若系统允许）、链路速率等信息

使用方式

在对话中用自然语言描述你的需求，AI 会自动调用本模块的功能。
