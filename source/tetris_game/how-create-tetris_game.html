模块概述

实现经典 Tetris（俄罗斯方块）：board 为 10 列 × 20 行，空位用 -1 表示；七种 tetromino 以预定义相对坐标数组 PIECES[][][][] 存储，字符标签 I,O,T,S,Z,L,J。当前块与预览块、得分、等级、消行数等由插件字段维护；通过 ModulePlugin.invoke 暴露游戏循环 API。

棋盘、方块与计分

  • 生成方块：spawnPiece() 将上一轮的 nextPiece 变为 currentPiece，再随机新的 nextPiece；初始位置约在顶部中央，若不能放置则 gameOver=true。
  • 碰撞检测：canPlace(piece, rotation, px, py) 遍历形状格子，检查边界与已锁定格（≠ -1）。
  • 锁定与消行：lockPiece() 将当前块写入 board（值为方块类型索引 0–6），调用 clearLines()：自下而上整行满则下沉复制，顶部清空；得分使用 lineScores[cleared] * level，其中 lineScores = {0,100,300,500,800}，cleared 截断至 4。
  • 等级：level = difficulty + linesCleared / 10（初始 level = difficulty）。
  • 旋转：尝试下一旋转态；若不可则尝试左右平移一格（简易 wall kick）。
  • Hard drop：drop() 下落到底，每下落一格 score += 2，然后 lockPiece()。

类 TetrisPlugin 方法一览

方法说明
invoke分发 startGame|move|rotate|drop|tick|getState|restart
startGame(int)难度 1–3，initGame()
initGame清空 board（-1），重置分数/消行/level=difficulty，spawnPiece
move(String)left|right|down；向下若阻塞则 lockPiece
rotate旋转 + 左右踢墙
drop / tick硬降 / 软降一格或锁定
buildStateJSON：score、level、linesCleared、currentPiece/nextPiece 字符、board 字符串（ASCII 画）
renderBoard锁定格为 █，空为 ·，当前活动块为 ▓（游戏结束时不再绘制活动块）

API Actions — 输入 / 输出

startGame

params：difficulty（1–3，默认 2）

output 字段：gameStarted，gameOver，score，level，linesCleared，difficulty，boardWidth（10），boardHeight（20），currentPiece，nextPiece，board（多行字符串）。

move

params：direction："left" | "right" | "down"

非法方向返回 success:false。

rotate · drop · tick

均无额外参数；需游戏已开始。 tick 模拟重力一步。

getState · restart

restart：若 difficulty==0 置 2；复用或新建 Random 后 initGame()。

成功响应统一为：success:true，output 为嵌套 JSON 字符串，可选 _displayText。

数据流

startGame → initGame → spawnPiece
    ↓
循环: move / rotate / drop / tick
    → canPlace? → 更新坐标或 lockPiece → clearLines → spawnPiece
    → gameOver 则后续操作仅返回当前状态（部分 action）

输出: buildState() → output 字符串 + formatDisplay()
