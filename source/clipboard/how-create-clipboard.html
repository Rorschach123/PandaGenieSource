模块概述

本模块结合 Android ClipboardManager 与本地 JSON 文件实现系统剪贴板读写、清空，以及剪贴板历史持久化、检索与清空。剪贴板访问必须在主线程执行，插件通过 Handler(Looper.getMainLooper()) + CountDownLatch 同步等待，超时 3000ms。

Capabilities： file_read, file_write, clipboard。权限声明： MANAGE_EXTERNAL_STORAGE（用于历史文件路径）。

架构与数据流

invoke(action, params) → 剪贴板类 action: runOnMain(ClipboardTask) → 主线程: ClipboardManager.getPrimaryClip / setPrimaryClip(ClipData.newPlainText) → 历史类 action: loadHistoryArray / appendHistoryEntry / saveHistoryArray → 文件: /sdcard/PandaGenie/data/clipboard/history.json ← ok(output JSON string, _displayText)

历史文件格式

路径常量：HISTORY_FILE = "/sdcard/PandaGenie/data/clipboard/history.json"

文件内容为 JSON 数组，按时间追加在末尾；每条为对象 { "text": "...", "timestamp": long }。HISTORY_MAX = 100 时从索引 0 删除最旧条目。getClipboardHistory 返回顺序为从新到旧（从数组尾部向前取）。

若文件损坏或解析失败，loadHistoryArray 返回空数组 []。

API Actions — 输入 / 输出

getClipboard

无参数。output： {"text":"..."}，无内容时为空串。

setClipboard

必填 text（可为空串）。output： {"set":true}。缺参抛 IllegalArgumentException → error JSON。

clearClipboard

写入 ClipData.newPlainText("", "")。output： {"cleared":true}。

getClipboardHistory

参数 limit：默认 20；≤0 时重置为 20；最大不超过 HISTORY_MAX。

output： items（JSONArray），count。

saveToHistory

读取当前剪贴板文本并 appendHistoryEntry。output： saved, textLength。

searchHistory

必填非空 keyword。在存储文本上 contains 子串匹配，不区分大小写（Locale.ROOT）。结果从新到旧。

output： items, count。

clearHistory

写入空数组文件。output： {"cleared":true}。

Java 类与方法详解

ClipboardPlugin implements ModulePlugin

内部接口 ClipboardTask

String run(Context ctx) — 在主线程执行的剪贴板读写闭包。

runOnMain(Context, ClipboardTask)

若已在主线程则直接执行；否则 mainHandler.post + CountDownLatch.await(3000ms)。超时抛出 IllegalArgumentException("Clipboard operation timed out")。

剪贴板底层
  • clipboardManager(Context) — 校验 CLIPBOARD_SERVICE
  • readClipText(ClipboardManager) — getPrimaryClip() 首项 getText()
  • getClipboard / setClipboard / clearClipboard
文件 IO
  • ensureParentDir(File)
  • readFileUtf8 / writeFileUtf8 — FileReader/FileWriter 平台默认编码（注释约定 UTF-8 兼容 JSON）
  • loadHistoryArray / saveHistoryArray / trimOldest / appendHistoryEntry
展示格式化常量

DISPLAY_BODY_MAX=2000, DISPLAY_ITEM_MAX=400, DISPLAY_SEARCH_PREVIEW=5 — 用于 format*Display 与 truncateForDisplay、mdCell。

invoke 异常

IllegalArgumentException → error(ex.getMessage())，与其它模块一致返回 JSON 而非向外抛出。
