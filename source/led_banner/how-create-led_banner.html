模块概览

根据文字、颜色、显示模式（scroll / fade / static）、字号、速度与特效关键字生成供 H5 渲染的 JSON，包含内联 CSS 片段（cssStyle、bgStyle、textShadow）。generateBanner 与 applyTemplate 成功时响应带 _openModule: true，提示宿主打开模块全屏页。

模板数据

  • CYBER_TEMPLATES：源码中定义 20 条赛博/霓虹配色 [名称, fontColor, bgColor]。
  • STAR_TEMPLATES：38 条明星/应援配色，结构相同。
  • applyTemplate：先精确匹配名称，再在两类表中做子串模糊匹配（用户输入小写包含于模板名）。找不到抛 Template not found。

特效与 CSS 生成

parseEffects 解析逗号/分号/空白分隔关键字，支持英文 glow/flash/shake/bold/italic 及中文触发「荧光」「闪」「抖」「粗」「斜」。glow 为真时生成多层 text-shadow 模拟霓虹。bold/italic 写入 cssStyle。注意：flash/shake 在 JSON effects 中为布尔开关，具体动画由前端 H5 消费。
