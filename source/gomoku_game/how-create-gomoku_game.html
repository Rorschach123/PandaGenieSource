模块概览

经典五子棋（Gomoku）：15×15 棋盘，玩家执黑（BLACK）先手，程序执白（WHITE）。内存态维护 board[][]、gameStarted、gameOver、winner（0 无 / 1 玩家 / 2 AI / 3 和棋）、moveCount。

规则与胜负判定

checkWin(row, col, player) 沿四个方向 DIRECTIONS = {(1,0),(0,1),(1,1),(1,-1)} 双向延伸计数，连续 ≥5 子即胜。棋盘满且未分胜负则 winner=3。

AI 策略（威胁检测 + 评分）

findAiMove 优先级：

  1. 白棋能成五则下（findThreat(WHITE,5) 用 checkWin）。
  2. 阻止黑棋五连（findThreat(BLACK,5)）。
  3. 白棋造四连（targetLen=4，countMaxLine≥4）。
  4. 挡黑四。
  5. 白三 / 挡黑三。
  6. 否则 findBestScoreMove：仅考虑距已有棋子 Chebyshev ≤2 的空位；evaluatePosition 用中心距离 + 假设黑白落子后的 countMaxLine 加权（白×10，黑×8）。同分随机打破。
