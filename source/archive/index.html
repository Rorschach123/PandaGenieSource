模块信息

版本: v--

引擎: libarchive_module.so

C++ native压缩解压引擎，支持ZIP(含密码加密)、TAR、GZ、TAR.GZ格式。

接口列表

接口说明参数
compressZipZIP压缩inputPaths, outputPath, password
decompressZipZIP解压archivePath, outputDir, password
compressTarTAR打包inputPaths, outputPath
decompressTarTAR解压archivePath, outputDir
compressGzGZ压缩inputPath, outputPath
decompressGzGZ解压archivePath, outputPath
compressTarGzTAR.GZ压缩inputPaths, outputPath
decompressTarGzTAR.GZ解压archivePath, outputDir
listContents列出压缩包内容archivePath
