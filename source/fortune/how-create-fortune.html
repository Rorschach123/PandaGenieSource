模块概览

本模块基于公历日期与可选姓名生成确定性伪随机（seeded）运势文案，包含 7 个 fortune level（大吉→大凶）、吉语词库、宜/忌、幸运数字与颜色，并附带 农历信息。仅供互动展示，无预测科学依据。

manifest：无额外 capabilities/permissions。pluginClass：ai.rorsch.moduleplugins.fortune.FortunePlugin。

运势生成算法

  • 种子：seed = dateSeed(y,m,d) + nameHash(name)，其中 dateSeed = year*10000 + month*100 + day；nameHash 为类乘法哈希（空名为 0）。
  • 等级：level = seededInt(seed, 7, 1) + 1 → 1–7，对应 LEVEL_NAMES_ZH / LEVEL_NAMES_EN / LEVEL_SYMBOLS。
  • 词条索引：seededInt 配合不同 salt（2–6）从 FORTUNE_WORDS[level-1]、LUCKY_COLORS、ADVICE_DO、ADVICE_DONT 取值。
  • 线性同余混合：mixed = seed * 6364136223846793005L + salt * 1442695040888963407L，再 abs % bound。

农历 Lunar Calendar

computeLunar 以 1900-01-31（UTC）为农历正月初一基准，用 LUNAR_INFO 整型位图数组推算：lunarYearDays、leapMonth、leapDays、monthDays。输出含 lunarYear/Month/Day、isLeapMonth、天干地支年、zodiac、lunarDateString 等。年份限制 1900–2100；早于基准日抛错。
